Limbajul HTML

HTML (HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html il gasiti in imaginea urmatoare care se afla la adresa http://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.

Imaginea are atributul height de valoare 147.

Tag-urile html cele mai utiliate in realizarea tabelelor sunt:

table defineste un tabel
tr un rand intr-un tabel
td o celula intr-un tabel

Observatie: pentru a pune contur unui tabel se poate (dar nu se recomanda) a se pune atributul border="1".

La validare va da eroarea: The border attribute on the table element is obsolete.

Formularele se pot crea cu ajutorul tag-ului form care poate contine tag-urile: input, textarea, label, select, option, button.

Pentru mai multe detalii consultati cursul sau tutorialul HTML Forms and Inputs de la adresa http://w3schools.com/html/html_forms.asp

Pentru a crea liste se pot folosi tag-urile:

  1. ul - listă neordonată
  2. ol - listă ordonată
  3. li - item din listă
  4. dl - listă cu descrieri
  5. dt - termen dintr-o listă cu descrieri
  6. dd - descrierea dintr-o listă cu descrieri

Pentru a insera un document HTML în altul se poate folosi tag-ul iframe.

Astfel, exemplul de mai jos insereaza conținutul de la adresa https://html.spec.whatwg.org/multipage/
